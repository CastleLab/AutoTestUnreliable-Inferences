
EgoHands Public - v1 specific
==============================

This dataset was exported via roboflow.ai on December 9, 2020 at 8:06 PM GMT

It includes 4800 images.
Hands are annotated in YOLO v3 Darknet format.

The following pre-processing was applied to each image:
* Auto-orientation of pixel data (with EXIF-orientation stripping)

No image augmentation techniques were applied.
